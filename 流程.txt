# PosterMind AI - 技术架构 & 数据库 & API & 页面原型 v1.0

---

## 一、系统概览

**系统模块**：

1. **前端**：用户界面，选择风格、输入文字、查看生成海报、微调、下载
   技术：Next.js + TailwindCSS + Shadcn UI

2. **后端**：业务逻辑、用户系统、风格系统、海报生成管理
   技术：FastAPI + Python

3. **数据库**：用户信息、海报模板、风格库、历史记录、会员信息
   技术：PostgreSQL

4. **缓存**：Redis，用于短期存储生成任务状态

5. **AI引擎**：

   * 文案 & Prompt生成：GPT-4o
   * 图像生成：Stable Diffusion / GPT Image / Flux Kontext

6. **对象存储**：Cloudflare R2 或 AWS S3，用于存储生成的海报图片

## 三、API接口文档

### 1. 用户注册 / 登录

| Method | URL                | Request                               | Response                    |
| ------ | ------------------ | ------------------------------------- | --------------------------- |
| POST   | /api/auth/register | `{username, email, password}`         | `{success, user_id}`        |
| POST   | /api/auth/login    | `{email, password}`                   | `{success, token, user_id}` |

## 五、AI生成工作流

1. 用户提交海报信息（文字 + 风格 + 模板）
2. 系统通过 Prompt Builder 生成 AI 背景生成 Prompt
3. AI生成背景图
4. 后端读取模板布局 + 文字渲染规则
5. 程序渲染文字到背景图上
6. 存储海报图片
7. 更新生成状态给前端
8. 用户微调 & 下载
